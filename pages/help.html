Xbox 360 Edition
Using Xenia Emulator

For playing Xbox 360 games on a PC you'll need to download Xenia Emulator. Extract it to a folder.
Then, you'll need to obtain the base game file. For legal issues we can not provide it here, a quick archive.org search should get you the game. All you need to do then is open up the emulator, and drag and drop the game file into the window. The game will start.

Installing Title Updates (TUs)

For installing updates, you'll need to download an old version of Xenia Canary, found here.
